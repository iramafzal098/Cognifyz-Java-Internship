# Cognifyz Java Internship

This repository contains the Java Development internship tasks completed during the Cognifyz Technologies Internship Program.

---

## Levels Completed
- Level 1
- Level 2

---

## Tasks Completed

### Level 1

#### Task 1 - Temperature Converter
A Java program that converts temperature between Celsius and Fahrenheit using user input and menu-driven operations.

**Skills:** Functions, conditional statements, loops, user input handling.

---

#### Task 2 - Palindrome Checker
A Java program that checks whether a word or phrase is a palindrome.

**Skills:** String manipulation, loops, conditional statements.

---

#### Task 3 - Student Grade Calculator
A Java program that calculates and displays the average grade of a student.

**Skills:** Loops, arrays, basic arithmetic operations.

---

#### Task 4 - Random Password Generator
A Java program that generates random passwords based on user-selected criteria.

**Skills:** Random number generation, string manipulation, user input handling.
