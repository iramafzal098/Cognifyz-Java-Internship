A menu-driven Java application that converts temperatures between Celsius and Fahrenheit.

* Methods
* Conditional statements
* Loops
* User input handling
* Basic calculations

**Concepts Practiced:**

* String handling
* Loops
* Conditional statements
* Character comparison

**Concepts Practiced:**

* Methods
